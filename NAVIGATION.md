## Adding New Projects

To add a new project to this portfolio:

1. **Create repository**: `git@github.com:pilakkat1964/[project-name].git`
2. **Add project entry**: Create `docs/_projects/[project-name].md`
3. **Update NAVIGATION.md**: Add project to this guide
4. **Update README.md**: Add project link to README
5. **Commit and push**: `git add -A && git commit -m "feat: add [project-name] project"`
